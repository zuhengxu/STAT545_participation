# Lists
## Bullet lists

```
* 1
* 2
* 3
```
or

```
- 1
- 2
- 3
```


- 1
- 2
- 3

Add space or Tab to create multi-level bullet lists：

```
- 1
 - 1.1
 - 1.2
  	 - 1.2.2
  	 - 1.2.3
- 2 
```

- 1
 - 1.1
 - 1.2
  	 - 1.2.2
  	 - 1.2.3
- 2 

 
## Number lists

```
1. 1
2. 2
3. 3
```


1. 1
2. 2
3. 3

When creating multi-level number lists，we can use bullet lists and numbers ：

```
- 1
 - 1.1
 - 1.2
  	 - 1.2.2
  	 - 1.2.3
- 2 
```


- 1. title 
	- 1.1 subtitle 
	- 1.2 subtitle 
        - 1.2.1 subtitle 
        - 1.2.2 subtitle 
- 2. title
